The Library

Brief

A social media experiment seen through the lens of book clubs! Create or join a club and share your thoughts with your friends.

Stack

Challenge

Creating a platform on which users can build a community. Where the information for that medium (a book) is readily, and easily available.

Solution

Rails API implementation more than just storage of users and club information.

JSON web token authorization to ensure users have appropriate access and clubs security.

Leverage two different sources so users can set the book their clubs or themselves are currently reading:

  1. New York Times Book API
  2. Google Books API

Live on Heroku

Order Up

Brief

Point of Sales (POS) prototype. Intended to create a fluid workflow for a retail or restaurant worker.

Stack

Challenge

Emulate the UX of a professional POS interface, while minimiziming time the end-user spends searching for items.

Solution

Create a layout a where Categories and Subcategories are easily distinguishable and editable.

Having a simple navigation means not only have a higher number of saleable items on screen, but ensuring the appropriate items are presented.

Shaving seconds off an action that occurs thousands of times a day.
